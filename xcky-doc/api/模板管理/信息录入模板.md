- 序号

#### lable

- lable表示录入项英文名（传递给后台使用）

#### name

- name表示录入项中文名（供页面展示使用）

#### isOpen

- 是否开启（1是0否）

#### must

- must表示录入项是否必填（1是0否）

#### readonly

- readonly代表是否只读（1是0否）
